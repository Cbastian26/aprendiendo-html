Etiquetas básicas de texto.

p: En los párrafos se escribe texto. Se puede escribir texto sin etiqueta alguna también, pero no es ni correcto ni conveniente.
b: Escribe en negritas.
i: Letras itálicas.
u: Subraya.
sub: H2O.
sup: E = mc2.
mark: Marca texto.
small: Letras pequeñas.
strong: Esta etiqueta define importancia en el texto.
em: Esta etiqueta define énfasis en el texto.
backquote y cite:

If you want to improve, be content to be thought foolish and stupid.
Epictetus.

Etiquetas de agrupación de contenidos.

br: Da saltos de línea.
hr: Define un salto semántico de contenido a traves de una línea horizontal.

Etiquetas de formateo.

pre: La etiqueta pre hace que se respete la indentación del texto escrita dentro.
code: Le da formato código al texto escrito dentro. 

            

            function messageMaker (type, style){
                return function(str){
                    console.log(`%c ${type}: ${str}`, style);
                };
            };

            const info = messageMaker(`Info`, `background: blue; color: white`);
            info(`Este es un ejemplo de como usar las etiquetas pre y code.`);

Etiquetas de estructuración (maquetación).

Es importante tener en cuenta que el correcto uso de las etiquetas de estructuración es subjetivo hasta cierto punto, adquiriendo la cualidad de ser siempre materia perfectible su aplicación en un sitio web.

div: Etiqueta sin carga semántica de estructuración de contenido. Antes de HTML5 era la única etiqueta estructural.
head: Etiqueta que contiene la cabecera del sitio web, o de una sección del sitio.
nav: Contiene los elementos que le permiten al usuario navegar por el sitio web.
main: Contiene el contenido principal del sitio web. Por SEO solo debe haber una etiqueta main por documento HTML.
aside: Contiene contenido complementario (secundario) en un sitio web, como la publicidad por ejemplo.
section: Contiene una sección de contenido.
article: Contiene una seccion de contenido el cual se explica por si mismo, sin depender del contenido del resto del sitio web.
footer: Define el pie de página de un sitio web o de una sección.
adress: Generalmente contiene una dirección o algun medio de contacto.

Etiquetas de Bloque vs Etiquetas de Línea.

Las etiquetas de bloque ocupan todo el largo de la linea en el contenedor donde se encuentran, independientemente del tamaño de su contenido, por lo que generan saltos de linea.
Las etiquetas en línea solo ocupan el espacio de acuerdo a su contenido, por lo que no necesariamente generan saltos de línea.

Estilos en HTML.

Hay 3 formas de añadir estilos a un documento HTML:

            1.-Estilos en línea: Usando el atributo style.
            
                <span style="background: rgb(120, 120, 196); color: white;"></span>
            

            2.-Estilos internos: Añadiendo una etiqueta style, preferiblemente en el head.
            
                <style>
                    span{
                        background: rgb(120, 120, 196); 
                        color: white;                       
                    }
                </style>
            

            3.-Estilos externos: Enlazando una hoja de estilos en el head. (Forma óptima).
            
                <link rel="stylesheet" href="estilosHTML.css">

Scripts en HTML.

Hay 2 formas de añadir scripts a un documento HTML:


        1.-Forma directa: Añadiendo una etiqueta script al documento HTML.
        
            <script>
                console.log(`Ejemplo de etiqueta script.`);
            </script>          
        

        2.-Forma indirecta: Enlazando un archivo contenedor del script. De preferencia al finalde la etiqueta body.
           (Forma Óptima)
        
            <script src="scriptsHTML.js"></script>
